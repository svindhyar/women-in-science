## Women in STEM Data Story

Read my Medium article here: https://medium.com/code-like-a-girl/what-data-actually-tells-us-about-women-in-science-f699c04ebadb

I made this small notebook project exploring how women are doing across STEM fields, countries, and years, using enrollment, graduation rate, and gender gap indicators.

This started as an analysis notebook, but turned into a mini data story:
Who’s improving? Where are the biggest drop-offs from enrollment to graduation? Which countries look most promising over time? etc were answered.

What I did here:
- Loads and checks the dataset
- Builds country-level and STEM-field summaries

Creates visual comparisons for:
- Female enrollment vs graduation
- Country-level differences
- Trends over time

Runs a few deeper checks:
- Correlation analysis
- Trend slope significance
- Effect size comparison (Country vs STEM Field)
- Cluster profiling
- Retention risk trend analysis
